# Downloader-Youtube

## Usando Python e as bibliotecas PySimpleGui & Pytube
## Depencencies
### Necessario para execução
```
sudo apt-get update
sudo apt-get install python3
```
## Run Aplication
### Se não estiver usando uma venv
```
python3 -m venv venv
```
### Ativar a venv
```
$ source venv/bin/activate
```
### Executando a aplicação
```
python main.py
```
